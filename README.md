# Air Quality Analysis

Analysis of air quality variations across urban areas for city planning.

## Installation

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Launch Jupyter:
```bash
jupyter notebook
```

3. Open `Air Quality Analysis.ipynb`

4. Update the CSV file path in Cell 2

5. Run all cells

## Requirements

Your CSV file needs these columns:
- City, Country, Date
- PM2.5, PM10, NO2, SO2, CO, O3
- Temperature, Humidity, Wind Speed

## What You'll Get

- 10 visualizations of air quality data
- Statistical analysis by city
- WHO guideline comparisons
- Executive summary with recommendations
